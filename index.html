Welcome

Teknologi adalah suatu hal yang unik bagi saya. Sejak kecil saya sangat menyukai seni. Panggil saja saya Mif, saya tinggal di daerah Jawa Timur. Semua yang berhubungan dengan seni saya suka. Seiring berjalannya waktu, pola kehidupan berkembang berbeda. Teknologi termasuk suatu seni bagi saya, teknologi dapat mengubah sesuatu menjadi lebih baik. Bahkan tidak hanya sesuatu, teknologi juga dapat mengubah pola kehidupan manusia. Teknologi diciptakan oleh manusia dengan tujuan salah satunya untuk mempermudah suatu pekerjaan manusia. Teknologi berkembang begitu cepat sampai manusia dapat kehilangan kontrol akan teknologi. Dampak positif dari teknologi sangat banyak, begitu juga sebaliknya dampak negatif dari teknologi tidaklah sedikit. Seni yang unik ini(Teknologi) harus digunakan dengan baik. Semua dampak atau efek yang dihasilkan dari teknologi tergantung dari orang yang menggunakan teknologi tersebut.


Jadi, gunakanlah teknologi dengan baik dan benar ya!

Tentang Saya

Saya senang membuat website, design, menggambar, membuat logo, terutama tentang programming. Saya sekarang masih dalam tahap belajar untuk menjadi lebih baik dan saya akan terus tetap belajar. Dulu saya tidak memiliki keinginan untuk bekerja di bidang IT. Pengetahuan saya tentang dunia IT juga masih kosong. Saya hanya suka bermain game tanpa ada rasa penasaran bagaimana game tersebut dibuat.


Seiring berkembangnya ekonomi di Indonesia, ternyata teknologi memiliki dampak yang besar akan hal tersebut. Saya menjadi sangat tertarik terhadap dunia IT. Saya mulai belajar tentang IT mulai dari dasar. Mengikuti beberapa training, sertifikasi gratis, dan program pembelajaran gratis salah satunya seperti program "IDCamp" yang diselenggarakan oleh Indosat Ooredoo Hutchison bekerja sama dengan Dicoding. Program ini sangatlah baik untuk masyarakat Indonesia, karena selain gratis program seperti ini dapat juga memberikan dampak posistif bagi yang ingin belajar tentang IT mulai dari dasar. Masyarakat dengan ekonomi menengah kebawah juga sangat terbantu dengan adanya program ini. Web ini saya buat salah satunya sebagai suatu kelengkapan tugas dari program ini. Saya harap dengan adanya program ini dapat memunculkan generasi baru yang dapat memberikan dampak positif bagi ekonomi digital di Indonesia.
